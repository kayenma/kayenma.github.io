How it Works!

Search for near by people who wants to go out bowling!


The app is going to ask the user to enable the GPS location to identify where is the user is located.

The app is going to give the user list of options to choose from such as (drink, bowling, cinema ..etc)

Once the user selects the activity, the app will prompt the user to take a selfie (live at the moment).

The user will be prompt to write few words (not required).

Timer will start 6 hours. After 6 hours, the selfie will be deleted

The user can wait to receive message from people near by to chat and meet.

The user also can see who is near by and what they like to do within 6 hours.
